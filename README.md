# Master Thesis Code

This repository holds the code for the Master thesis "Establishment of a mechanically heterogeneous microenvironment to study its impact on growth factor signaling in bone progenitor cells.". 

Python code for [*Spiral deformation analysis*](https://github.com/yaeraw/Master_Thesis/blob/18d1cfd5733922ee0b03361e604fa9bd174e47df/Spiral_Code/Spiral_deformation_analysis.ipynb) and [*Spiral subtraction for cell counting*](https://github.com/yaeraw/Master_Thesis/blob/346310a4018af4c070a5b4b1c7715fe90e57b343/Spiral_Code/Spiral_Subtraction_Code.ipynb).

Image J Macros for [*nuclear pSMAD1/5/9 quantification*](https://github.com/yaeraw/Master_Thesis/tree/18d1cfd5733922ee0b03361e604fa9bd174e47df/nuclear_pSMAD_quantification).

R code for [*single cell RNA seq analysis*](https://github.com/yaeraw/Master_Thesis/blob/18d1cfd5733922ee0b03361e604fa9bd174e47df/single_cell_RNA_seq/Analysis_of_single_cell_seq_data.R).
